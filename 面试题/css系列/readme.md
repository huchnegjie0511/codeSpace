# 一、说说你打对css盒模型的理解
1. 是什么？（说出css盒模型概念）
浏览器在页面布局时，将所有的元素/容器表示为一个个矩形盒子，每一个盒子包含四个部分：content（内部容器）border 边框 margin外边框 padding内边框

2. 标准盒模型
盒子总宽度：width ＋content +border+ margin+ padding
