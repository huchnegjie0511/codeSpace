# react
- vue 和react的相同点和不同点
    - 都是mvvm架构的现代前端框架，react早于Vue
    - VDOM diff 组件化

    - Vue上手简单 主要得益于良好的API
    - REACT JSWAY 
        - 组件就是一个函数，返回html 就是组件
        - jsx   
        dom 操作 单独为ReactDOM 提供
    - vue支持指令  v-model 双向数据绑定  
        react 认为双向绑定功能不好，不支持
    - 为什么要用ts？
        - JS是弱引用类型  ts添加了类型系统
        - 减少很多错误出现  
        - 不需要的代码
            需求改变后，有些代码没有删除
        - 类型推导
        - 静态编译
            出了错误还不知道， 反复的调试 打错了一个字母
        - ts更适合大型多人协作项目开发
            带着脚铐跳舞
